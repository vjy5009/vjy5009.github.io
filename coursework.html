Operating System Projects

Thread Schedulers

Implemented 4 thread scheduling algorithms for a uni-processor environment. The 4 thread scheduling algorithms to implement are:

  • First-Come First-Serve (FCFS)
  • Shortest Run-Time First (SRTF)
  • Priority-Based Scheduling (PBS)
  • Multi-Level Feedback Queue (MLFQ)

Memory Management

Implemented 4 memory allocation schemes to mimic malloc() and free() functions for a single thread system. The 4 memory allocation schemes to implement are:

  • First-Fit (FF)
  • Best-Fit (BF)
  • Worst-Fit (WF)
  • Buddy System

Virtual Memory Paging

Emulated a virtual memory management scheme with 2 different page replacement algorithms. The 2 page replacement algorithms to implement are:

  • First-In First-Out (FIFO)
  • Clock Replacement (Second-Chance)

System Programming Project

RAID Device Driver

Implemented a basic tagline device driver to translate high-level tagline operations into low-level RAID operations, where a tagline is a storage abstraction similar to a data file. It allows support of multiple taglines, multiblock reads and writes, block-level redundancy, disk failure recovery, block level cache, and network communication to a RAID device server.
